AudioFlow Studio Mac 使用说明

一、推荐打开方式
1. 解压压缩包，不要直接在压缩包预览里打开。
2. 先双击 Open_AudioFlow.command。
3. 弹出终端窗口后，等待它自动打开 AudioFlow Studio。
4. 如果系统询问是否允许终端打开软件，选择允许。

二、如果还是提示无法验证
1. 不要点“移到废纸篓”。
2. 按住 Control 键点击 AudioFlow Studio，选择“打开”，弹窗里再次选择“打开”。
3. 如果仍然无法打开，进入“系统设置 - 隐私与安全性”，找到 AudioFlow Studio，选择“仍要打开”。

三、基本使用
1. 打开软件后先完成卡密激活。
2. 导入需要处理的音频，选择方案后开始处理。
3. 处理完成后到桌面 AudioFlow_Output 文件夹查看成品文件。
4. 不要手动打开或修改 AudioFlow Studio.app 里面的 Contents 文件夹。

四、常见问题
1. 提示软件已损坏或打不开：删除当前解压出来的软件，重新解压压缩包后先运行 Open_AudioFlow.command。
2. 打开后没有反应：确认系统版本为 macOS 12 或更高，重启电脑后再打开。
3. 无法激活：检查网络是否正常，确认卡密没有多输入空格。
4. 处理失败或没有输出：把歌曲文件名改短一点，只保留中文、英文或数字后再导入。
5. 老款 Intel 苹果电脑如果打不开，请联系售后获取 Intel 专用版本。
