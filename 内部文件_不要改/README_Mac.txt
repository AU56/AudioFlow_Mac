AudioFlow Studio Mac 使用说明

一、首次打开
1. 解压压缩包后，建议先把 AudioFlow Studio 拖到“应用程序”文件夹。
2. 第一次打开时，如果提示“Apple 无法验证 AudioFlow Studio”，请不要点“移到废纸篓”。
3. 按住 Control 键点击 AudioFlow Studio，选择“打开”，弹窗里再次选择“打开”。
4. 如果仍然无法打开，进入“系统设置 - 隐私与安全性”，找到 AudioFlow Studio，选择“仍要打开”。

二、基本使用
1. 打开软件后先完成卡密激活。
2. 导入需要处理的音频，选择方案后开始处理。
3. 处理完成后到输出目录查看成品文件。
4. 不要手动打开或修改 AudioFlow Studio.app 里面的 Contents 文件夹。

三、常见问题
1. 提示无法验证：按住 Control 键点击软件，选择“打开”。
2. 提示软件已损坏或打不开：删除当前解压出来的软件，重新解压压缩包后再按上面的方式打开。
3. 打开后没有反应：确认系统版本为 macOS 12 或更高，重启电脑后再打开。
4. 无法激活：检查网络是否正常，确认卡密没有多输入空格。
5. 处理失败或没有输出：把歌曲文件名改短一点，只保留中文、英文或数字后再导入。
6. 输出目录找不到：默认在桌面 AudioFlow_Output 文件夹。
