AudioFlow Studio Mac 使用说明

第一步：先解压这个压缩包。
第二步：双击打开 AudioFlow Studio.app。
第三步：不要进入 Contents 文件夹，也不要手动点击 Contents/MacOS 里面的文件。

如果提示无法打开：
1. 右键 AudioFlow Studio.app，选择“打开”。
2. 如果仍被拦截，打开“系统设置 - 隐私与安全性”，找到 AudioFlow Studio，点击“仍要打开”。

如果软件闪退、打不开、无法激活：
请把报错截图、macOS 系统版本、卡密发给售后处理。
